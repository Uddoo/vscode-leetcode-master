# Support

Use the repository issue tracker for LeetCode Master support requests.

## Before Opening An Issue

- Update to the latest extension version.
- Run `LeetCode Master: Refresh` from the LeetCode Explorer.
- Confirm whether the issue happens on `leetcode.com`, `leetcode.cn`, or both.
- Check whether the same action works in a browser on the selected endpoint.

## What To Include

- VS Code version.
- LeetCode Master extension version.
- Operating system.
- Active endpoint: `leetcode` or `leetcode-cn`.
- Node.js version and whether `leetcode.nodePath` is configured.
- Steps to reproduce the issue.
- Relevant output logs from VS Code Output panel.

Do not include passwords, session cookies, tokens, or full solution code if it is private.
